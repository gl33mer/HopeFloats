This folder contains all of the elements needed to package your Chrome Extension.

Submitting updates to the extension requires you to zip all of the files in this directory and upload to your Chrome developer account.

Full documentation of Chrome Extensions available here: http://developer.chrome.com/extensions/getstarted.html

Note: this current extension is using v1 of the manifest. All new versions of floathope will require you to upgrade to the V2 manifest.

Documentation of upgrading from v1 to v2 available here: http://developer.chrome.com/extensions/tut_migration_to_manifest_v2.html
